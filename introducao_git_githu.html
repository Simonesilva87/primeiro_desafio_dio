Introdução ao Git e ao GitHub

O que é o Git e GitHub?

O Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.
Já o GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo.

Lista de atalhos do Git;

ls = Lista;
cd + nome da pasta = Abre a pasta;
crt + l = Limpa o terminal;
mkdir + nome da pasta = Cria nova pasta;
cd .. = Volta um nível de pasta;

Lista de comando do Git

Para comitar

git add * e git commit -m "uma frase coerente com o arquivo"

Monitorar status dos arquivos

git status

comando para conferir configurações do Git

git config --list

Alterar nickname e email no Git

git config --global --unset user.email
git config --golbal -- unset user.nickname
git config --global user.email (colocar mesmo email do GitHub)
git config --global user.nickname (colocar mesmo nickname do GitHub)

Para empurrar o Arquivo

git push origin master

Para adicionar a origem

git remote add origin (link do novo repositório do git hub>

Para mostrar a lista dos repósitorios

git remote -v
